## Проект: Автотест для API сервиса Яндекс Самокат

Этот проект сделан для проверки работы API.  
Автотест проверяет создание заказа и получение данных по треку заказа.

### Что делает тест
1. Создаёт заказ через POST-запрос.  
2. Получает номер трека заказа.  
3. Проверяет, что по этому треку можно получить данные о заказе.  
4. Убеждается, что код ответа сервера равен **200**.

### Файлы проекта
- `create_and_get_order_test.py` — основной тест.  
- `configuration.py` — настройки и пути к API.  
- `sender_stand_request.py` — запросы к API.  
- `requirements.txt` — зависимости.  
- `.gitignore` — исключения для гита.
