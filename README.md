## Описание проекта
Проект включает SQL-запросы и автотест для проверки работы API Яндекс Самокат.

## Структура репозитория
- **create_and_get_order_test.py** — автотест: создание заказа и проверка его по треку.  
- **main.py** — тест с комментарием о выполнении задания.  
- **sql_task1_in_delivery.sql** — SQL-запрос: логины курьеров и количество их заказов в доставке.  
- **sql_task2_order_statuses.sql** — SQL-запрос: количество заказов по статусам.  
- **final_test_result.png** — скриншот выполнения теста с именем и когортой.

## Запуск автотеста
1. Активировать виртуальное окружение:
   ```bash
   & .venv/Scripts/Activate.ps1
   ```
2. Запустить тест:
   ```bash
   pytest create_and_get_order_test.py -v
   ```
3. Проверить результат в консоли (ожидается `1 passed`).

## Логи
Все логи записываются в `/var/www/backend/logs/error.log`.
